BodyPump Instructor Cheatsheet

Quick, repeatable cues to help remember choreography for every release. Starting with BodyPump 135.

1-Hour Memorization Sprint

Fast Recall Tips

  • Create a 3-word hook for each track (e.g., “row-clean-press”).
  • Anchor the first 2 timings in every block (4/4 → 3/1 → 1/1).
  • Say the move out loud on the first 8-count, then count it.
  • Chunk the release: Warm-up+Squats, Chest+Back, Triceps+Biceps, Lunges+Shoulders+Core.
  • End each track with a “finish cue” you always say.

60-Minute Plan

  1. 0–10: Read all track hooks + key timings once.
  2. 10–25: Warm-up + Squats (speak timings over music).
  3. 25–40: Chest + Back + Triceps.
  4. 40–50: Biceps + Lunges.
  5. 50–60: Shoulders + Core + Cooldown, then run hooks from memory.
